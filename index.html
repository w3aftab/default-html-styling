Default HTML Styling

Intoduction

Default HTML Styling(DHTMLS) is a link of basic stylesheet which can be used to style your website only using html elements. You do not have to worry about css coding. You just have to write the html code. it will autometically style most of the html elements. If you want to start web designing and you know HTML but you do not know CSS, you can start with DHTMLS.

Copy the link tag and past it onto your html head element:

<link rel="stylesheet" href="https://w3aftab.github.io/default-html-styling/basic.min.css"/>
